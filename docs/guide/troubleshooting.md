# 문제 해결

## `실행` 버튼을 눌렀는데 오류가 나옵니다

**가능한 원인**

- 팝업에 JDoodle `Client ID/Secret`이 저장되지 않음
- JDoodle 일일 호출 한도 초과(HTTP 429)
- 코드 컴파일 에러

**확인 순서**

1. 확장 아이콘 클릭 후 키가 저장되어 있는지 확인
2. 저장 후 다시 `실행`
3. 계속 실패하면 출력 메시지(컴파일 에러/서버 오류) 확인

## 제출이 실패합니다

**가능한 원인**

- BOJ 세션 만료
- Cloudflare 토큰(`cf-turnstile-response`) 만료
- 일시적인 네트워크/서버 문제

**해결**

1. 페이지 새로고침 후 다시 시도
2. BOJ 로그인 상태 재확인
3. 동일 문제 반복 시 브라우저 콘솔 에러 확인

## 수정 제출에서 코드가 비어 보입니다

확장은 `/submit/<problemId>/<submissionId>` 경로와 query/form/source 링크를 통해 원본 코드를 복원합니다.

복원이 늦는 경우가 있어 일정 시간 재시도를 수행하므로, 페이지 진입 직후 즉시 빈 코드여도 잠시 대기해 주세요.

## 테마/코드/테스트 케이스가 유지되지 않습니다

저장 데이터는 `chrome.storage.local`을 사용합니다.

- 브라우저 데이터 정리 정책
- 확장 재설치
- 동일 프로필 미사용

위 조건에서는 데이터가 유지되지 않을 수 있습니다.
